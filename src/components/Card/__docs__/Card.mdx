# Card

Card component with different props.

#### Example

<Canvas of={Card.Primary} />

## Usage

```ts
import {Card} from "sld-ui";

const Example = () => {
  return (
      <Card
        size={"small"}
        text={"Card"}
        onClick={()=> console.log("Clicked")}
        primary
      />
  );
};

export default Example;
```

#### Arguments

- **text** _`() => void`_ - A string that represents the text content of the Card.
- **primary** - A boolean indicating whether the Card should have a primary styling or not. Typically, a primary Card stands out as the main action in a user interface.
- **disabled** - A boolean indicating whether the Card should be disabled or not. When disabled, the Card cannot be clicked or interacted with.
- **size** - A string with one of three possible values: "small," "medium," or "large." It defines the size or dimensions of the Card.
- **onClick** - A function that is called when the Card is clicked. It receives a MouseEventHandler for handling the click event on the Card element.
